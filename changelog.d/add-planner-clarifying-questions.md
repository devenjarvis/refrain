### Added

- The planner Sonnet subprocess can now pause mid-draft and ask the developer a clarifying question. When the task description is genuinely ambiguous, the planner calls a new `ask_user` MCP tool; refrain spawns `refrain planner-question-server` as the bridge and surfaces the question in the plan editor as a new question mode. `enter` submits the typed answer and the planner resumes; `esc` skips the question with an empty answer so the planner falls through to a best-guess assumption rather than deadlocking. Per-draft unix sockets and a planner.Server keep the bridge isolated to the in-flight draft and torn down when it exits.
