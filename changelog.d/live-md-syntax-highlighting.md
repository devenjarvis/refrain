### Added

- The plan editor (`internal/tui/planeditor`) now renders ANSI-styled markdown live in both scroll and edit modes. Headers, bold/italic/inline-code spans, blockquotes, list bullets, horizontal rules, and fenced code blocks (per-language chroma, plaintext fallback) are highlighted as the user types. Long lines soft-wrap to terminal width with list continuations indented to the bullet column rather than re-bulleted. Scroll-mode and edit-mode wrap at the same column, so the `i`/`esc` mode toggle is visually continuous. The chroma style is hardcoded to `monokai` for now, mirroring `internal/tui/diff/render.go`; configurable themes are deferred. Implemented as two new packages: `internal/tui/mdrender` (chroma-backed renderer) and `internal/tui/mdtextarea` (a thin `bubbles/v2 textarea.Model` wrapper that overrides only `View()`).
